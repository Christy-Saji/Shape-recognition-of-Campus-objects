# ShapeVision — Shape Classification of Campus Objects

A **Digital Image Processing (DIP)** project that classifies campus object images into geometric shapes — **Circle**, **Rectangle**, or **Triangle** — using a combination of classical DIP techniques and deep learning.

---

## ✨ Features

- 🧠 **MobileNetV2** transfer learning for shape classification
- 🔬 **Canny Edge Detection** (DIP) visualized side-by-side with the original image
- 🌐 **Flask web app** with a dark, modern UI
- 📊 **Evaluation metrics** tab — accuracy, per-class precision/recall/F1, confusion matrix
- ⚡ Drag & drop image upload with animated confidence results

## 🔬 DIP Techniques Used

| Technique | Where | Purpose |
|---|---|---|
| **Grayscale Conversion** | `flask_app.py` | Reduces image to intensity channel for edge processing |
| **Gaussian Blur** | `flask_app.py` | Smooths noise before edge detection |
| **Canny Edge Detection** | `flask_app.py` | Extracts structural shape boundaries |
| **Image Resizing / Interpolation** | `flask_app.py`, `train.py` | Scales images to 224×224 for model input |
| **Intensity Normalization** | `flask_app.py`, `train.py` | Scales pixel values from [0,255] → [0,1] |

## 🚀 Setup & Run

### 1. Create and activate virtual environment
```bash
python -m venv DIP
# Windows
DIP\Scripts\activate
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train the model *(skip if `best_model.h5` already exists)*
```bash
python train.py
```

### 4. Generate evaluation metrics *(populates the Metrics tab)*
```bash
python eval.py
```

### 5. Start the Flask app
```bash
python flask_app.py
```

Open **http://localhost:5000** in your browser.
